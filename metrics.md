# ðŸ“ˆ 1. Cumulative Return (Total Return)

**Definition:**  
The total percentage gain/loss over the entire backtest.

**Formula:**  
\[
\text{Cumulative Return} = \prod_{t=1}^{T}(1 + r_t) - 1
\]

**Interpretation:**  
Higher is better.  
Shows overall profitability.

# âš–ï¸ 3. Sharpe Ratio

**Definition:**  
Return relative to volatility.

**Formula:**  
\[
\text{Sharpe} = \frac{R - R_f}{\sigma}
\]

Where:  
- \( R \) = Strategy return  
- \( R_f \) = Risk-free rate (e.g., US T-Bills)  
- \( \sigma \) = Standard deviation of returns

**Interpretation:**  
- > 1.0 â†’ Good  
- > 1.5 â†’ Very good  
- > 2.0 â†’ Institutional-grade

# ðŸ“‰ 4. Sortino Ratio (Downside-Risk Sharpe)

**Definition:**  
Sharpe but penalizes only *downside* volatility.

**Formula:**  
\[
\text{Sortino} = \frac{R - R_f}{\sigma_{\text{downside}}}
\]

**Interpretation:**  
More realistic than Sharpe for ML strategies.  
A strategy may have high volatility but good upward moves â€” Sortino reflects that.

# ðŸ“‰ 5. Max Drawdown (MDD)

**Definition:**  
Largest peak-to-trough loss.

**Formula:**  
\[
\text{MDD} = \frac{\text{Min Equity Curve Value} - \text{Max Prior Peak}}{\text{Max Prior Peak}}
\]

**Interpretation:**  
Lower absolute value is better.  
Example:  
- -10% = very good  
- -25% = acceptable  
- -50% = high risk

# ðŸ“† 6. CAGR (Compounded Annual Growth Rate)

**Definition:**  
Annualized return over the entire backtest.

**Formula:**  
\[
\text{CAGR} = (1 + R)^{\frac{1}{\text{years}}} - 1
\]

**Interpretation:**  
Normalizes performance across strategies with different backtest durations.
